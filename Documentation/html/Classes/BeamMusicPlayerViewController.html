Overview

The Music Player component. This is a drop-in view controller and provides the UI for a music player. It does not actually play music, just visualize music that is played somewhere else. The data to display is provided using the datasource property, events can be intercepted using the delegate-property.

Instance Methods

next

Skips to the next track.

- (void)next

Discussion

If there is no next track, this method does nothing, if there is, it skips one track forward and informs the delegate. In case [BeamMusicPlayerDelegate musicPlayer:shouldChangeTrack:] returns NO, the track is not changed.

previous

Skips to the previous track.

- (void)previous

Discussion

If there is no previous track, i.e. the current track number is 0, this method does nothing, if there is, it skips one track backward and informs the delegate. In case the [BeamMusicPlayerDelegate musicPlayer:shouldChangeTrack:] returns NO, the track is not changed.
