BeamMusicPlayerDataSource Protocol Reference

Conforms to NSObject
Declared in BeamMusicPlayerDataSource.h

Overview

The DataSource for the BeamMusicPlayerViewController provides all data necessary to display a player UI filled with the appropriate information.

musicPlayer:artworkForTrack:receivingBlock:

Returns the artwork for a given track.

- (void)musicPlayer:(BeamMusicPlayerViewController *)player artworkForTrack:(NSUInteger)trackNumber receivingBlock:(BeamMusicPlayerReceivingBlock)receivingBlock

Parameters

player

the BeamMusicPlayerViewController that needs artwork.

trackNumber

the index of the track for which the artwork is requested.

receivingBlock

a block of type BeamMusicPlayerReceivingBlock that needs to be called when the image is prepared by the receiver.

Discussion

The artwork is returned using a receiving block ( BeamMusicPlayerReceivingBlock ) that takes an UIImage and an optional error. If you supply nil as an image, a placeholder will be shown.
